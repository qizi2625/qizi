设计（论文）题目：
基于Spring Boot和LLM的智能房屋租凭系统的设计与实现
设计（论文）的主要内容：
本毕业设计基于 SpringBoot + Vue + MyBatis + MySQL 的前后端分离应用架构，主要实现如下功能：系统用户分权管理（基于RBAC模型实现租客、房东、管理员等多角色精细化权限控制）、房源信息发布与管理、在线租赁流程以及智能化功能。智能化功能以个性化推荐为核心，通过构建轻量级用户画像来实现：系统持续采集用户的收藏等关键行为，并从中提取房源特征标签（如 近地铁、精装修），动态构建并更新用户的偏好标签集合；推荐引擎通过计算该偏好标签集合与房源标签的实时匹配度，生成精准的“猜你喜欢”列表。相应系统模块包括：权限管理、用户管理、房源信息与标签管理、租赁订单管理、用户行为采集与画像构建模块、智能推荐引擎模块、智能客服（LLM集成）模块、消息通知模块、全文检索（Elasticsearch）模块等模块。后端采用 SpringBoot 框架进行开发，采用 Redis 作为缓存中间件，用于热点数据缓存与会话管理，以及 JWT 进行用户身份信息的保存和验证等。
